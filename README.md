## 5. Miniguia de Estudos

### Resumo
O panorama da análise de dados em 2026 consolida a linguagem Python como a "língua franca" da área, mas com uma mudança drástica na performance e no papel do profissional. O ecossistema evoluiu de ferramentas clássicas de manipulação para bibliotecas de alta performance (como Polars e DuckDB) e integração profunda com Inteligência Artificial. O analista moderno deixou de gastar 70% do seu tempo em limpeza manual para focar em validação estratégica, interpretação de modelos e tradução de números em narrativas acionáveis. Este documento sintetiza as principais ferramentas, fluxos de trabalho e requisitos educacionais que definem o setor atualmente.

### Glossário
Tipos de Dados:
int (Inteiro): Números sem casas decimais (ex: 10, -5)
.
float (Flutuante): Números decimais com vírgula (ex: 10.5)
.
str (String): Cadeias de caracteres usadas para textos, sempre entre aspas
.
bool (Booleano): Valores lógicos que podem ser apenas Verdadeiro (True) ou Falso (False)
.
